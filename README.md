PROJECT OVERVIEW:
</br><br>As a first-year student at VIT Bhopal, I’ve stepped into the role of a Student Digital Ambassador. This isn't just a title; it’s a responsibility to help my batchmates navigate the digital world safely and professionally. To kick things off, I’ve built a Digital Literacy Portfolio that serves as a roadmap for my peers.My work is organized into a single GitHub repository, which acted as my primary learning tool for version control and digital organization. The project is divided into five practical tasks that cover the essentials of modern student life: 1.Visual Communication: I designed an infographic to break down why digital literacy actually matters in our daily lives. 2.Professional Identity: I set up foundational profiles on platforms like LinkedIn and GitHub to start building a career-ready online presence. 3.Technical Skills: I practiced coding challenges on platforms like HackerRank and used Google Workspace to create collaborative tools like a digital awareness quiz. 4.Professional Etiquette: I drafted formal emails for professors and internship coordinators to master the "unspoken rules" of professional communication. 5.Cyber Safety: Finally, I researched real-world threats like UPI fraud to create a prevention guide, helping my peers stay one step ahead of scammers.This project has transformed from a set of tasks into a personal toolkit that I’ll use throughout my four years of engineering.
